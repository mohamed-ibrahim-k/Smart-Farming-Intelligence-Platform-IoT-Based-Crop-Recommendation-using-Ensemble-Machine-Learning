# Smart-Farming-Intelligence-Platform-IoT-Based-Crop-Recommendation-using-Ensemble-Machine-Learning
IoT-enabled precision agriculture platform integrating ESP32, Raspberry Pi, RS485 soil sensors, ThingSpeak, weather APIs, and ensemble ML (CatBoost, XGBoost, LightGBM &amp; Gradient Boosting) for real-time, AI-driven crop recommendations using live soil and environmental data.
